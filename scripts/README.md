# Prism Development Scripts

Quick development scripts for Prism.

## Available Scripts

### `./scripts/clean-db.sh`
Clean the Prism database (useful for testing from scratch).

### `./scripts/build.sh`
Build the Prism app and show only errors/warnings.

### `./scripts/test.sh`
Run database tests.

### `./scripts/run.sh`
Build and open Prism in Xcode.

## Usage

All scripts should be run from the project root:

```bash
./scripts/clean-db.sh
./scripts/build.sh
./scripts/test.sh
./scripts/run.sh
```
